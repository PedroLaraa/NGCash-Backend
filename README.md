# NGCash Backend

Instruções para rodar o projeto:
 
 - É necessário ter o Docker e o NodeJs instalados localmente!!!
 - Inicie um novo terminal ;
 - Execute o comando "docker-compose up -d" ;
 - A porta alocada localmente é a "3030" ;

##

End-Points:

- ``` POST: /user ```
- ``` POST: /login ```
- ``` POST: /transaction ```
- ``` GET: /profile ```
- ``` GET: /transaction```
- ``` GET: /balance```
